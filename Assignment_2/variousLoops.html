Different Types of Loops:


  1. For Loop:

    for (var i = 1; i <= 10; i++) {
     /* Do something with the value of i here, maybe print it......*/
    }


  2. While Loop:

    var i = 1;
    while (i <= 10) {
     /* Do something with the value of i here, maybe print it......*/
     i++;
    }


  3. Do-While Loop:

    var i = 1;
    do {
     /* Do something with the value of i here, maybe print it......*/
     i++;
    } while (i <= 10);
